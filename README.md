## How to generate the APK file

### Option 1: Build Online (Fastest, No PC needed)
We have included a GitHub Actions workflow. You can build this app online for free!
1. Unzip this folder on your computer.
2. Create a new repository on **GitHub** (https://github.com).
3. Upload all the unzipped files into that repository. **IMPORTANT: Make sure you also upload the hidden `.github` folder!** If you just drag-and-drop on a Mac/Windows, it might ignore hidden folders.
4. **DO NOT** click "Configure Java/Gradle workflow" on GitHub. The correct workflow is already in the `.github/workflows` folder you uploaded.
5. Click on the **"Actions"** tab in your GitHub repository to see the build running.
6. The build process will start automatically. Once completed, scroll down and download the **my-android-app-apk** zip file from the Artifacts section. Inside, you will find your APK!

### Option 2: Build with Android Studio (Local PC)
1. Extract this ZIP file.
2. Download and install **Android Studio** if you haven't already: https://developer.android.com/studio
3. Open Android Studio, click **"Open"**, and select the extracted folder.
4. Wait a moment for Android Studio to sync the Gradle build.
5. From the top menu, go to **Build > Build Bundle(s) / APK(s) > Build APK(s)**.
6. Once completed, a popup will appear at the bottom right. Click **"locate"** to find your `.apk` file.

You can now copy this `.apk` file to your Android phone and install it!
